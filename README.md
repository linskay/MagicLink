# MagicLink — Подсистема Источников и Автоконфигурации

Этот модуль предназначен для автоматического поиска, фильтрации и подготовки VPN/Proxy конфигураций.

## Основные возможности
- **Авто-апдейт**: Загрузка данных из GitHub (Raw, Repos, Releases) и по ссылкам подписок.
- **Pipeline обработки**:
    - Валидация (хост, порт, протокол).
    - Удаление дублей.
    - Тест задержки (TCP ping < 500ms).
    - Тегирование (страна, источник).
- **Хранение**: Локальная база данных SQLite.
- **Выход**: Генерация готовых JSON-конфигураций для `sing-box`.

## Структура проекта
- `core/`: Основная логика на Java + Maven.
- `magiclink.db`: Локальная база данных узлов.

## Использование
Запуск основного класса `com.magiclink.core.MagicLinkCore` инициирует обновление всех источников и подготовку актуального списка узлов.
